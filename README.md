# plcn

## Install git and gitlab-runner
./install.ps1

## Connect gitlab-runner to gitlab project

1. gitlab project set
2. gitlab-runner register

```
Edit config.toml
.\gitlab-runner.exe stop
.\gitlab-runner.exe start
```
